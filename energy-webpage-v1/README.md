# COS30045 – Data Visualisation
## Exercise 0.2 – Appliance Energy Consumption Website

---

## Summary of Work Done

This project involved building a small interactive website about appliance energy consumption in the Australian market. The website was built using HTML, CSS, and JavaScript, and is hosted on GitHub.

### Pages built
- **Home** – Overview of household energy consumption in Australia, including key facts and statistics
- **Televisions** – Detailed information about TV energy consumption by type and size, including a cost comparison table
- **About Us** – Information about the project, developer, data source, data processing, and ethics

### Features implemented
- Three-page website with JavaScript navigation (no page reloads)
- Top navigation menu with active page highlighting
- Mouse hover feedback on navigation links
- Power logo in navigation that returns user to home page
- Cards with icons for visual presentation
- Statistics section with key energy facts
- Data table showing estimated annual TV running costs
- Footer with year, name, and GenAI acknowledgement
- CSS styling with green colour scheme matching the energy theme
- Responsive layout using CSS Grid

### Tools and technologies used
- HTML5
- CSS3
- JavaScript
- GitHub Desktop for version control
- Visual Studio Code for development
- Live Server extension for local testing

## GenAI Declaration

### Tools used
- **Claude (Anthropic)** – Used via claude.ai
- **GitHub Copilot** – Available through GitHub Education

### How GenAI was used
GenAI tools were used to assist with the following aspects of this project:

- Generating the initial HTML structure and page layout
- Writing and formatting CSS styles including the navigation bar, cards, stats section, and table
- Writing JavaScript for the page navigation and active link highlighting
- Suggesting content and text for the energy consumption topic
- Debugging issues such as CSS not loading due to incorrect file naming
- Explaining how to set up the folder structure and GitHub workflow

### Reflection
Using GenAI tools significantly sped up the development process, especially for repetitive tasks like writing CSS and setting up the HTML structure. However, it was important to review and understand every piece of code generated, as some suggestions required modification to work correctly in this specific project.

For example, the CSS file was initially named incorrectly and the link in the HTML had to be fixed manually. This showed that GenAI does not always account for the specific file structure of a project and requires careful review.

Overall, GenAI was a useful assistant but not a replacement for understanding the code. All generated code was reviewed, tested, and modified where necessary.
